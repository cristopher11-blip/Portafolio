Juego Interactivo

React Native / React / Libreria Phaser

Fecha: 18/02/2024

Juego Interactivo fue el resultado de 5 meses de trabajo para crear un video juego con el objetivo de enseñar el funcionamiento de React y React Native y la libreria Phaser. Desde crear niveles ya diseñados, hasta patrones avanzados de nevegación, este proyecto ha sido el ejemplo de futuros creadores de video juegos con el fin de que las personas se interesen a la programacion de una forma interactiva y dinamica
